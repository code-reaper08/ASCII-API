ASCII-API

A simple JSON fake API, Which serves ASCII values for Alphanumeric characters.

Access URL:

https://my-json-server.typicode.com/code-reaper08/ASCII-API/ is the main URL used to access the below routes.

Available Routes :

ASCII-API contains 3 routes to access the static data.

  • AlphaC

    This Route gives the ASCII code for Capital Alphabets on request.
    Example
    https://my-json-server.typicode.com/code-reaper08/ASCII-API/AlphaC?name=R

    gives this ,

    [ { "id": 19, "name": "R", "ASCII": "082" } ] as a response.

  • AlphaS

    This Route gives the ASCII code for small Alphabets on request.
    Example
    https://my-json-server.typicode.com/code-reaper08/ASCII-API/AlphaS?name=u

    gives this ,

    [ { "id": 48, "name": "u", "ASCII": "117" } ] as a response.

  • Num

    This Route gives the ASCII code for Numbers on request.
    Example
    https://my-json-server.typicode.com/code-reaper08/ASCII-API/Num?name=8

    gives this ,

    [ { "id": 62, "name": "8", "ASCII": "056" } ] as a response.
